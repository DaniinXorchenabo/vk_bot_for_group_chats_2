/erease - очистить память для вашей беседы
/gen - сгенерировать новое сообщение
/stat - показать статистику по текущей беседе
/help - список команд
/kw - установить кнопки
/no_kw - убрать кнопки
/stat_me - получить статус пользователя
Чтобы я ответил на ваш вопрос, напишите: "Бот, <ваш вопрос>?"
----------
/log_in <пароль> - залогиниться как администратор, однако, чтобы пользоваться командами администратора, необходимо войти в сессию (/sign_in). Данную команду необходимо ввести только одиин раз
/sign_in - войти в сессию администратора (до этой команды бот будет считать вас обычным пользователем)
/sign_out - выйти из сессии администратора (после этой команды бот будет считать вас обычным пользователем), чтобы войти обратно введите /sign_in
/add_admin <id: только цифры> - добавляет админа с указанным id
/del_me - удалит тебя из админов
добавить возможность рассылки постов или сообщений по беседам
-----------
/dev_log_in <пароль> - залогиниться как разработчик, однако, чтобы пользоваться командами разработчика, необходимо войти в сессию (/dev_sign_in). Данную команду необходимо ввести только одиин раз
/dev_sign_in - войти в сессию разработчика (до этой команды бот будет считать вас обычным пользователем)
/dev_sign_out - выйти из сессии разработчика (после этой команды бот будет считать вас обычным пользователем), чтобы войти обратно введите /dev_sign_in
/add_developer <id: только цифры> - добавляет разработчика с указанным id
/add_admin <id: только цифры> - добавляет админа с указанным id
/dev_del_me - удалит тебя из разработчиков
/del_admin <id: только цифры> - удаляет админа с указаным id
/get_list_admins - выводит список админов из памяти и из БД
/get_list_developers - выводит список разработчиков из памяти и из БД
/edit_admin_password <пароль> - позволяет менять пароль администратора, при этом удаляет всех админов
